# 🧮 Maths Beyond Classroom (MBC)

**Maths Beyond Classroom (MBC)** is a full-stack Olympiad Mathematics learning platform designed to help students prepare for competitive examinations such as **IOQM, RMO, INMO, SEHSS, NMTS**, and other mathematics competitions through structured learning, practice, and assessment.

---

## 🚀 Current Status

**Active Development**

The platform has evolved from a frontend prototype into a full-stack application with backend integration, cloud-based asset management, and an admin content management system.

---

# ✨ Features Implemented

## 🎨 Frontend

- Responsive UI across desktop, tablet, and mobile
- Modern homepage with premium design
- Student Dashboard
- Topic Explorer
- Study Materials Library
- Previous Year Questions (PYQs)
- Problem of the Day (POTD)
- Mock Tests page
- Pro Plans page
- About Platform pages
- Admin Dashboard

---

## ⚙️ Backend

- REST APIs using Express.js
- Prisma ORM integration
- CRUD operations for:
  - Problem of the Day
  - Study Materials
  - Previous Year Questions
  - Mock Tests
- Backend validation
- Database integration

## ☁️ Cloud Integration
